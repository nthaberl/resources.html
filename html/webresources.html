Web Development Resources

A list of useful websites and tools for reference.

Home
  • Mozilla Developer Network
    ★★★★★

    The gold standard resource for web development documentation. Provides a wealth of information, thorough documentation, and even provides free courses.

  • JavaScript Cheatsheet
    ★★★★☆

    An interactive quick reference guide for JavaScript basics, syntax, and functions. Information is provided on cards that can be moved around or hidden. Also provides other cheatsheets that are useful for web development like HTML, CSS, and even SEO.

  • Flexbox Labs
    ★★★★☆

    A visualization tool used to better understand CSS Flexbox properties and how they work in a layout. Provides templates for popular layouts and also provides visualization for CSS Grid. Can even generate code snippets based on the layout created.

  • Flexbox Froggy
    ★★★★★

    A gameified approach to learning CSS Flexbox! Help Froggy and friends by using Flexbox properties to align them on their lily pads.

  • CSS Diner
    ★★★★★

    Another gameified approach! Use CSS selectors to select the correct food items on the table. Increases in difficulty quickly as you progress through the levels.

  • Guide to JavaScript LocalStorage
    ★★★☆☆

    Quick breakdown for learning the use of JavaScript LocalStorage. Also provides quick overview of sessionStorage and cookies for client-side storage options and understanding the use cases, benefits, and disadvantages of all three approaches.

  • Traversy Media
    ★★★★★

    YouTube channel with thorough tutorials covering a wide range of web development tools and frameworks.

  • GitHub Tutorial by Karl Broman
    ★★★★☆

    Simple guide to get up and running with GitHub and using git in the command line. Goes over basic commands and workflow. Great for quick lookup without getting bogged down in technical documentation.
